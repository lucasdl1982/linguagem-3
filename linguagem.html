Conceitos Básicos da Web

HTML

O HTML (HyperText Markup Language) é a linguagem de marcação que estrutura o conteúdo de uma página. Ele define títulos, parágrafos, links, imagens e outros elementos. 

<h1>Olá Mundo</h1>
<p>Este é meu primeiro parágrafo em HTML</p>

CSS

O CSS (Cascading Style Sheets) é responsável pela aparência visual da página. Com ele, controlamos cores, fontes, espaçamento e até layouts responsivos. 

p {
  color: blue;
  font-size: 18px;
  text-align: center;
}

JavaScript

O JavaScript é a linguagem de programação que adiciona interatividade às páginas. Com ele podemos manipular elementos, responder a ações do usuário e integrar dados dinâmicos. 

<button onclick="alert('Olá Mundo!')">
Clique aqui
</button>
